Download PVR.MythTV

PVR.MythTV is written in C/C++. It should be possible to build it for nearly all XBMC/Kodi supported platforms. Prebuilt binaries are available for OpenELEC, OS X and Windows.

PVR.MythTV is licensed under GNU General Public License. For all of the gory details, read the file debian/copyright in the source distribution.

Latest stable version is 1.12.24 for Kodi 14 (Helix).
