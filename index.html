MythTV addon for XBMC~Kodi mediacenter

Sources ZIP File
Sources TAR Ball

MythTV frontend supports backend 0.27 up to 0.28. Allows streaming of Live TV & Recordings, listening to Radio channels, EPG and Timers.

Written in C/C++, it uses the library CPPMyth.
